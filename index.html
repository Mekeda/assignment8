Calculator

About This App

This is a basic calculator app that allows for switch control functionality to perform calculations.

Instructions

List of Modes:
  • Base Mode - Users are able to select any button and click it using the mouse or input numbers via the test box.
  • Arrow Key and Enter Functions - Users are able to use the arrow keys and the enter key to perform calculations. Using either arrow key buttons will highlight the first function on the calculator, in this case 7. When pressing the right arrow, the highlighted selection will move right of the current seletion, in this case 8. If the current highlighted selection is at end of a row upon pressing right arrow, the selection will move to the next row. To insert the desired selection, press the enter key when over the highlighted selection. Navigation can be achieved regardless of the key that is pressed. Using the up/down arrow keys will only allow the user to navigate the respective column when trying to select a highlighted selection.

    The following are the key combinations that were tested for switch control:

    • Two Keys - User can use only right (left) arrow and enter to use this calculator.
    • Three Keys - User can use right (left) arrow, enter and any other arrow keys (up, down, left) to use this calculator.
    • Four Keys - User can use right key, left key, enter key and any of the other directional key (up, down) to navigate this calculator.
    • Five Keys - User can use all the four arrow keys and enter to naviagate this calculator.
